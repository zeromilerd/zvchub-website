ZVC: Zero Vulnerability Computing

Problem

Cybercrime losses to global economy

https://cybersecurityventures.com/annual-cybercrime-report-2017/

Cybersecurity experts believe fool proof computer security is impossible

Two mandatory computer design rules make computers vulnerable

Third-Party Permissions
In-Computer Data Storage




Is it possible to circumvent these rules & achieve zero vulnerability

Solution

Our findings suggest Zero Vulnerability Computing (ZVC) may be possible

ZVC can be achieved by:


Banning all third-party permissions to non-native apps
Implementing In-Computer Offline Data Storage (ICOS)


Following practical implications of ZVC need further research:

Potential of building quantum resistant computers
Introducing solid-state-robustness and energy efficiency with Solid State Software on a Chip (3SoC) concept
